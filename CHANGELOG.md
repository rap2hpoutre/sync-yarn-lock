# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-09

### Added

- Initial release
- Sync `package.json` versions with `yarn.lock` for Yarn 4 (Berry)
- `--dry-run` flag to preview changes without modifying files
- `--silent` flag for quiet operation
- Colorful terminal output with progress indicators
- Support for `dependencies`, `devDependencies`, and `peerDependencies`
- Graceful handling of git/workspace dependencies
